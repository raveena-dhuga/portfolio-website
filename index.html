Travel List App

A React travel resource that is powered by Fetch API and REST Countries API

  • Features type ahead to begin to autocomplete the country or capital searched for
  • Displays further information including currency and languages spoken.
  • Uses controlled inputs to check items to be saved, which persist in local storage.
  • Shift + click will select multiple saved items, in either direction
  • Deselecting saved items works using the same method
  • Deleting saved items also updates them to unsaved on the main page using Context API and state management
